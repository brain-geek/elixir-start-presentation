Pattern matching

      # Lists
      iex> list = [1, 2, 3]
      iex> [1, 2, 3] = list
      [1, 2, 3]
      iex> [] = list
      ** (MatchError) no match of right hand side value: [1, 2, 3]

Pattern matching

      iex> [1|tail] = list
      [1, 2, 3]
      iex> tail
      [2, 3]
      iex> [2|_] = list
      ** (MatchError) no match of right hand side value: [1, 2, 3]
